# ANTIGRAVITY | Premium MERN Portfolio

A cinematic, production-ready full-stack portfolio website for a creative software development agency.

## Tech Stack

- **Frontend**: React, Vite, Tailwind CSS, Framer Motion, GSAP, Socket.io-client
- **Backend**: Node.js, Express, MongoDB, Socket.io, JWT
- **Animations**: GSAP for scroll-based storytelling, Framer Motion for UI interactions

## Features

- 🚀 Cinematic Hero Section with Parallax
- 📽️ Storytelling-based scrolling experience
- 🌗 Dark/Light Theme Switching
- 🔐 Secure Admin Dashboard (JWT Auth)
- ⚡ Real-time updates via Socket.io
- 📱 Fully Responsive & Premium UI/UX

## Quick Start

### 1. Prerequisites

- Node.js (v18+)
- MongoDB (Local or Atlas)

### 2. Backend Setup

```bash
cd backend
npm install
# Create .env file based on .env.example
npm start
```

### 3. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
